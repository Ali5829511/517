📋 الخطوات الأساسية - Basic Steps:
═══════════════════════════════════════════════════════════════════════════

1️⃣ تثبيت XAMPP - Install XAMPP
   ├─ حمّل من: https://www.apachefriends.org/download.html
   ├─ ثبّت في: C:\xampp (Windows) أو /opt/lampp (Linux)
   └─ تأكد من عمل Apache

2️⃣ تثبيت Python 3.11+ - Install Python 3.11+
   ├─ حمّل من: https://www.python.org/downloads/
   ├─ ✅ حدد "Add Python to PATH" أثناء التثبيت
   └─ تحقق: python --version

3️⃣ نسخ المشروع - Copy Project
   ├─ انسخ مجلد المشروع الكامل إلى:
   │  Windows: C:\xampp\htdocs\housing-system
   │  Linux:   /opt/lampp/htdocs/housing-system
   └─ انسخ محتويات هذه الحزمة إلى نفس المجلد

4️⃣ تثبيت المكتبات - Install Dependencies
   ├─ افتح Command Prompt / Terminal
   ├─ cd C:\xampp\htdocs\housing-system
   └─ pip install -r requirements.txt

5️⃣ إنشاء ملف .env - Create .env File
   ├─ أنشئ ملف جديد اسمه .env
   └─ أضف المحتوى التالي:
   
   OPENAI_API_KEY=your-api-key-here
   FLASK_ENV=production
   SECRET_KEY=change-this-to-random-string
   DATABASE_PATH=housing_database.db
   HOST=127.0.0.1
   PORT=5000

6️⃣ تشغيل النظام - Run the System
   
   🪟 Windows:
   ────────────
   انقر نقراً مزدوجاً على:
   scripts\start_flask_windows.bat
   
   🐧 Linux:
   ────────────
   ./scripts/start_flask_linux.sh

7️⃣ الوصول للنظام - Access the System
   ├─ افتح المتصفح: http://127.0.0.1:5000
   ├─ اسم المستخدم: admin
   └─ كلمة المرور: Admin@2025

🐛 حل المشاكل الشائعة - Common Issues:
═══════════════════════════════════════════════════════════════════════════

❌ Python not found
   الحل: أضف Python إلى PATH

❌ Port 5000 already in use
   الحل: غيّر PORT=8000 في ملف .env

❌ Module not found
   الحل: pip install -r requirements.txt

❌ Database not found
   الحل: تأكد من وجود housing_database.db في المجلد

❌ Apache not working
   الحل: أغلق Skype أو أي برنامج يستخدم المنفذ 80
